### ✅ 6. 失败重试机制
**状态**: 已实现
**验证位置**: `src/llm_chat/scheduler/task_executor.py:43-103`

**证据**:
```python
retry_count = 0
max_attempts = task.max_retries + 1  # 默认 max_retries = 3

while retry_count < max_attempts:
    try:
        # 执行任务
        result = self._execute_xxx(task)
        return execution  # 成功返回
    except Exception as e:
        last_error = str(e)
        retry_count += 1
        if retry_count < max_attempts:
            delay = min(self.base_delay * (2 ** (retry_count - 1)), self.max_delay)
            time.sleep(delay)  # 指数退避
```

**重试逻辑**:
- 最大重试次数: 3 次（可在任务中配置）
- 退避策略: 指数退避（base_delay * 2^(retry_count-1)）
- 最大延迟: 60 秒
