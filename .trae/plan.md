# 外部数据获取能力方案分析计划

## 背景

当前项目 **Vermilion Bird** 是一个 Python LLM 对话客户端，支持多种 API 协议（OpenAI、Anthropic、Gemini）。需要为其添加连接外部 MCP 工具的能力，并提供图形化配置界面。

## 需求明确

用户希望：
1. **作为 MCP 客户端** - 连接和使用外部 MCP 服务器提供的工具
2. **图形化配置** - 通过 GUI 界面配置和管理 MCP 工具连接
3. **示例支持** - 暂时以天气查询作为示例

### 核心组件

#### 1. MCP 客户端管理器 (`mcp_manager.py`)
- 管理多个 MCP 服务器连接
- 动态发现和注册工具
- 处理工具调用请求
- 支持连接状态监控

#### 2. MCP 配置模块 (`mcp_config.py`)
- MCP 服务器连接配置
- 支持多种传输方式（stdio、SSE、HTTP）
- 配置持久化

#### 3. GUI 配置界面
- MCP 服务器列表管理
- 添加/编辑/删除服务器配置
- 连接状态显示
- 工具列表预览

## 配置示例

```yaml
# config.yaml
mcp_servers:
  - name: "weather"
    command: "npx"
    args: ["-y", "@modelcontextprotocol/server-weather"]
    enabled: true
    
  - name: "brave-search"
    command: "npx"  
    args: ["-y", "@modelcontextprotocol/server-brave-search"]
    env:
      BRAVE_API_KEY: "your-api-key"
    enabled: true
```
